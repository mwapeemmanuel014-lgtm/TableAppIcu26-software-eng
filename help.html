📚 Frequently Asked Questions

What is this app?

The Timetable App is a personal schedule planner designed to help you:

  • View your complete weekly timetable in one place
  • Identify class clashes and scheduling conflicts
  • Find free time slots in your schedule
  • Print or export your timetable for offline use
  • Search for specific courses quickly

How do I search for a course?

Go to the Timetable page and use the search box at the top. Type the course name, code, or venue (e.g., "Advanced Physics" or "Main Hall") to filter the timetable.

What do the symbols mean?

  • ❌ = Confirmed clash (you cannot attend both courses)
  • ⚠️ = Potential overlap or warning (verify with department)
  • 📍 = Day/location marker

How do I resolve a class clash?

Visit the Clash Analysis page for detailed resolution strategies. Generally, you should:

  1. Contact your academic coordinator immediately
  2. Ask about alternative session times
  3. Check if an online or recorded version is available
  4. Consider deferring lower-priority electives
  5. Request academic accommodations if needed

Can I print my timetable?

Yes! Go to the Timetable page and click the 🖨️ Print button. Or use your browser's print function (Ctrl+P / Cmd+P) to save as PDF.

Can I filter by day or time?

The current version allows you to search by course name and venue. For advanced filtering, use your browser's find function (Ctrl+F / Cmd+F) to search for specific days or times.

Is my data private?

Yes. This app runs entirely in your browser—no data is sent to external servers or stored online. Your timetable information remains on your device only.

🔧 Troubleshooting

The app isn't loading properly

Try the following:

  1. Refresh the page (F5 or Ctrl+R)
  2. Clear your browser cache (Settings → Clear browsing data)
  3. Try a different browser
  4. Disable browser extensions that might interfere

Search isn't working

Ensure JavaScript is enabled in your browser. Check your browser's developer console (F12) for error messages. Try using exact course names or venue codes.

Print looks wrong

In the print dialog:

  • Set margins to "Minimal"
  • Disable headers/footers
  • Use landscape orientation for better spacing
  • Scale to fit page width
